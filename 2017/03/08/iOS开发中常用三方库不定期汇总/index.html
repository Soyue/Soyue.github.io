iOS开发中常用三方库不定期汇总

发表于 2017-03-08 | 分类于 iOS开发 | | 阅读次数

列举一些开发中常用的第三方框架，以后会不定期更新，鉴于目前开发还是以OC为主，Swift版本放在以后。

OC

  • AFNetworking
    AFNetworking是一个非常方便的网络请求库，可以轻松实现各种网络请求，比如经常使用的GET请求、POST请求，以及上传多张图片等。
  • SDWebImage
    目前最受欢迎的图片下载第三方框架，使用率很高。
  • Masonry
    轻量级的布局框架，拥有自己的描述语法，采用链式语法封装自动布局，简洁明了，具有很高的可读性。
  • MJExtension
    字典和模型之间互相转换的超轻量级框架。
  • MJRefresh
    用法简单的上下拉刷新控件。
  • SVProgressHUD
    实现多种HUD效果。多用于程序正在执行耗时较长命令，需要用户等待。除了显示等待的HUD，还可以显示命令执行成功或者失败的HUD。
  • WMPageController
    类似于网易新闻客户端首页的效果，管理多个viewController，用于在scrollView上滑动，来管理多个相关联的viewcontroller。
  • RDVTabBarController
    定制化的TabBarController库，可动画显示隐藏tabbar栏以及个性化定制tabbar栏。
  • SDCycleScrollView
    无限循环自动图片轮播器。
  • IQKeyboardManager
    自动处理键盘事件的第三方库，可以很容易地解决弹起键盘遮盖输入框的问题，并且易于集成，不需要侵入性地继承特定的类，也不需要实现某些奇怪的协议，满足大部分需求只需要少于5行代码。

Swift

待续。。。
