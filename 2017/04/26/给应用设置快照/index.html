修改应用快照

发表于 2017-04-26 | 分类于 iOS开发 | | 阅读次数

当应用进入后台时,系统会给app生成当前界面的快照,此时,如果我们双击home键进入后台,就可以看到app的快照.然而,在某些情况下生成的快照并不符合我们的预期,比如,一些金融类的app,会泄露用户的隐私.(银行卡账号信息,资金信息,用户信息…可能都会在快照中发现).比较常见的做法是给生成的快照覆盖一层”毛玻璃”,这样就看不见了🙈

主要思路

  • 在- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(nullable NSDictionary *)launchOptions中,注册观察UIApplicationWillResignActiveNotification和UIApplicationDidBecomeActiveNotification的通知.
  • 在程序即将退出时创建一个UIWindow对象,”覆盖”在屏幕上;程序活跃状态时,隐藏该window,程序即将退出时hidden属性为NO.
  • 对该window添加UIVisualEffectView视图对象(iOS 8之后有效,使之拥有毛玻璃效果).
