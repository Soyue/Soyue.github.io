微信小程序开发相关准备事宜

发表于 2018-02-07 | | 阅读次数

梳理了一下小程序开发需要准备的一些相关事宜,结合之前产品设计的相关经验教训,主要从账号的申请,后台服务器配置以及UI设计图这三个方面出发,目的是方便产品的快速开发上线.

小程序账号申请

参照微信官方文档点击https://mp.weixin.qq.com/wxopen/waregister?action=step1根据指引填写相关信息并提交相应资料.

这里列出几条注意点

  • 每个邮箱仅能申请一个小程序。
  • 企业类型帐号可选择两种主体验证方式。 方式一：需要用公司的对公账户向腾讯公司打款来验证主体身份。打款信息在提交主体信息后可以查看到。 方式二：通过微信认证验证主体身份，需支付300元认证费。认证通过前，小程序部分功能暂无法使用。
  • 主体信息一旦确认不可变更。

UI设计的一些建议

  • 官方建议开发微信小程序时设计师使用iPhone6作为视觉稿的标准。
  • 在较小的屏幕上不可避免的会有一些毛刺,请在开发时尽量避免这种情况。
  • 个人建议UI设计图上传至蓝湖或者其他类似的设计图协作平台上,以便开发者准确的获取相关标注信息。
  • 切图尽量以png的格式给出。
